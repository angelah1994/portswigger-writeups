# SQL Injection

This directory contains write-ups for SQL Injection labs completed on the PortSwigger Web Security Academy.

SQL Injection is one of the most common and impactful web application vulnerabilities. It occurs when user-controlled input is incorporated into SQL queries without proper validation or parameterization, allowing attackers to manipulate database queries.

These write-ups demonstrate practical techniques for identifying, exploiting, and understanding SQL injection vulnerabilities in a controlled lab environment using Burp Suite and manual testing.

## Skills Practiced

- Identifying SQL Injection vulnerabilities
- Boolean-based Blind SQL Injection
- Error-based SQL Injection
- UNION-based SQL Injection
- Enumerating database information
- Extracting sensitive data
- Using Burp Suite Proxy
- Using Burp Suite Intruder
- HTTP Request Manipulation
